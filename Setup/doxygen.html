Linux

$ # Ubuntu, Debian
$ sudo apt-get install doxygen doxygen-doc doxygen-gui graphviz mscgen cppcheck
$ # Fedora
$ sudo yum install -y doxygen doxygen-doxywizard graphviz mscgen cppcheck

Windows, Mac OSX

  1. Download and install Graphviz

  2. Download and install doxygen

  3. Optional: Download and install mscgen

On MAC OSX you can install mscgen using homebrew using brew install mscgen

Doxywizard

You may run the standalone doxywizard to configure and invoke doxygen.

Eclox Eclipse Plugin

  1. Launch Eclipse and select Help → Install New Software …​

  2. In the pop-up dialog select Add …​ and enter ECLOX and http://download.gna.org/eclox/update

  3. Follow through the wizard

  4. To create a new doxyfile configuration select File → New → Other …​

  5. In the pop-up dialog select Other → Doxyfile

  6. Double-click the doxyfile in the Project Explorer to launch the wizard

  7. When done click the @ button on the toolbar to run doxygen
